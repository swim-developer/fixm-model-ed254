# swim-fixm-model-ed254 — Knowledge Base


## What This Is

**JAXB data model for FIXM 4.3 / EUROCAE ED-254.** Used by `swim-ed254-consumer`, `swim-ed254-provider`, and their validators for XML marshalling/unmarshalling of Arrival Sequence messages.

Generated via XJC (JAXB) from the ED-254 XSD schemas. Do not hand-edit generated classes.

## Usage

```xml
<dependency>
    <groupId>com.github.swim-developer</groupId>
    <artifactId>swim-fixm-model-ed254</artifactId>
</dependency>
```

## Build

```bash
mvn clean install -DskipTests
```
